`clpy`
------

`clpy` provides a low-level Cython_ binding to the Clp_ linear programming solver.

It is mostly useful when you want to specify (possibly very large) problems directly in matrix form, instead of using
higher-level interfaces like PuLP_.


.. _Clp: https://projects.coin-or.org/Clp
.. _Cython: http://cython.org/
.. _PuLP: https://github.com/coin-or/pulp
